# Note-Taking App

A simple note-taking app built with React. This app allows users to add, view, search, and delete notes. The app saves the notes locally in the browser using `localStorage`.

## Features

- Add a note with a title and description.
- View a list of all notes.
- Search notes by title.
- Delete notes from the list.
- Persistent data storage using `localStorage`.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **React Router DOM**: A routing library for React to handle navigation between components.
- **Tailwind CSS**: A utility-first CSS framework for styling the application.
- **localStorage**: A built-in browser feature to store data persistently in the client's browser.

## Libraries and Versions

- **React**: `18.2.0`
- **React Router DOM**: `6.6.1`
- **Tailwind CSS**: `3.2.7`
- **React Scripts**: `5.0.0`
